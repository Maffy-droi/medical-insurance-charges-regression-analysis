Project Overview 
This project applies Simple and Multiple Linear Regression techniques to predict medical insurance 
charges using demographic and lifestyle variables. 
The analysis was conducted using the Medical Cost Personal Dataset as part of a Regression Analysis  
The goal is to help an insurance company: 
• Set fair premiums 
• Identify high-risk clients 
• Manage financial risk 
• Improve pricing strategies
